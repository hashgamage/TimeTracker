**Time Tracker Application**

**Features Implemented**
Single Form CRUD: Create, Read, Update, and Delete time entries from a single screen.

Seeded Lookups: People and Tasks are pre-loaded as per requirements.

Validation: * Client-side validation using Angular Reactive Forms.

Server-side validation using C# Data Annotations (Decorators).

Architecture: * Repository Pattern for data access.

Dependency Injection for the EF Context.

Transactional updates to ensure data integrity.
